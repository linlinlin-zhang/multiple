# Plan 05: Fallback & Action Feedback

## Objective
Display inline action feedback cards in the chat sidebar after canvas actions execute. When the user says "做个计划" and a `create_plan` card appears on the canvas, the chat message should show a clickable feedback card: "已创建 plan 卡片：XXX" — clicking it focuses the corresponding node on the canvas. This closes the loop between chat command and canvas result, making the system feel responsive and reliable.

## must_haves
- After `applyVoiceActions` executes, each successful action's `nodeId` is returned to the chat message
- Chat messages display feedback cards for executed actions
- Feedback cards show action type icon, localized label, and card title
- Clicking a feedback card focuses the canvas on the corresponding node
- Feedback cards use the styling defined in UI-SPEC.md
- Works for all action types including rich nodes and zoom

```js
async function applyVoiceActions(value) {

}
```

```js
// BEFORE
if (message.role === "assistant" && message.actions?.length) {
  const actions = document.createElement("div");
  actions.className = "chat-action-summary";
  actions.textContent = t("chat.actionsApplied", { count: message.actions.length });

// AFTER
if (message.role === "assistant" && message.actionResults?.length) {
  message.actionResults.forEach((ar) => {
    const card = document.createElement("div");
    card.className = "chat-action-feedback";
    card.setAttribute("role", "button");
    card.setAttribute("tabindex", "0");

    const actionType = ar.type || "";
    const nodeId = ar.result?.nodeId || ar.nodeId || "";
    const title = ar.title || ar.result?.title || ar.nodeName || "";

    const icon = ACTION_FEEDBACK_ICONS[actionType] || "⚡";
    const labelKey = `chat.actionFeedback.${actionType}`;
    const label = t(labelKey) || t("chat.actionApplied");

    card.innerHTML = `
      <span class="chat-action-icon">${icon}</span>
      <span class="chat-action-label">${label}</span>
      <span class="chat-action-title">${escapeHtml(title)}</span>
    `;

    if (nodeId) {
      card.title = t("chat.clickToFocus") || "点击跳转到画布节点";
      card.onclick = () => focusNodeById(nodeId);
      card.onkeydown = (e) => { if (e.key === "Enter" || e.key === " ") { e.preventDefault(); focusNodeById(nodeId); } };
    }

    line.appendChild(card);
  });
}
```
