   - Export `useHistory(limit = 20, offset = 0, includeDemo = false)`:

   - Use plain `fetch`; no external data-fetching library needed.

| T-02-02 | Information Disclosure | /api/sessions/:id | accept | Session data is owned by single user (no auth); acceptable for self-hosted v1 |
| T-02-03 | DoS | Large limit param | mitigate | Backend clamps limit to 100; hook defaults to 20 |
