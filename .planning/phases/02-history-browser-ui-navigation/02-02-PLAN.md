## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser -> /api/sessions/:id | Session data includes user-generated content (chat messages, image hashes) |
| Browser -> /api/assets/:hash | Direct file serving; hash must be validated |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-04 | Information Disclosure | AssetDetailPane showing file hashes | accept | Hash is SHA-256 content hash, not sensitive; acceptable for self-hosted v1 |
| T-02-05 | DoS | Large image in detail pane | mitigate | Use native `<img>` with browser's natural lazy loading; backend serves files with immutable cache headers |
| T-02-06 | XSS | Chat message content rendered in detail pane | mitigate | Render chat content as text nodes (not innerHTML); React's JSX escaping handles this automatically |
</threat_model>
