Create `src/api/share.js` with two exported handlers.

`handleCreateShare(sessionId, res)` — POST /api/sessions/:id/share:
1. Validate `sessionId` is a non-empty string. If not, return 400 `{ error: "sessionId is required" }`.
2. Fetch the full session from Prisma using `prisma.session.findUnique` with `include: { nodes: true, links: true, assets: true, chatMessages: { orderBy: { createdAt: "asc" } } }`.
3. If session not found, return 404 `{ error: "Session not found" }`.
4. Build a `snapshotData` object that is a **self-contained, decoupled copy** of the session state:
   ```javascript
   const snapshotData = {
     title: session.title,
     isDemo: session.isDemo,
     createdAt: session.createdAt,
     viewState: session.viewState,
     nodes: session.nodes.map(n => ({
       nodeId: n.nodeId,
       type: n.type,
       x: n.x,
       y: n.y,
       width: n.width,
       height: n.height,
       data: n.data,
       collapsed: n.collapsed
     })),
     links: session.links.map(l => ({
       fromNodeId: l.fromNodeId,
       toNodeId: l.toNodeId,
       kind: l.kind
     })),
     assets: session.assets.map(a => ({
       hash: a.hash,
       kind: a.kind,
       mimeType: a.mimeType,
       fileSize: a.fileSize,
       fileName: a.fileName
     })),
     chatMessages: session.chatMessages.map(m => ({
       role: m.role,
       content: m.content
     }))
   };
   ```
   Note: `assets` stores only `hash` metadata — the actual image files are content-addressable in `storage/` and do NOT need duplication.
5. Create a `ShareToken` record via `prisma.shareToken.create` with `{ data: { sessionId, snapshotData } }`. Let Prisma auto-generate the `token` UUID.
6. Return 200 JSON:
   ```javascript
   {
     ok: true,
     shareUrl: `/share/${shareToken.token}`,
     token: shareToken.token,
     createdAt: shareToken.createdAt
   }
   ```

`handleGetShare(token, res)` — GET /api/share/:token:
1. Validate `token` is a non-empty string. If not, return 400 `{ error: "token is required" }`.
2. Query `prisma.shareToken.findUnique({ where: { token }, select: { token: true, snapshotData: true, createdAt: true, expiresAt: true } })`.
3. If not found, return 404 `{ error: "Share not found" }`.
4. If `expiresAt` is set and `new Date() > expiresAt`, return 410 `{ error: "Share link has expired" }`.
5. Return 200 JSON:
   ```javascript
   {
     ok: true,
     token: shareToken.token,
     createdAt: shareToken.createdAt,
     expiresAt: shareToken.expiresAt,
     snapshot: shareToken.snapshotData
   }
   ```

Both handlers must use the same `sendJson` helper pattern as `src/api/sessions.js`:
```javascript
function sendJson(res, status, data) {
  res.writeHead(status, {
    "Content-Type": "application/json; charset=utf-8",
    "Cache-Control": "no-cache"
  });
  res.end(JSON.stringify(data));
}
```

Import `prisma` from `../lib/prisma.js`.
