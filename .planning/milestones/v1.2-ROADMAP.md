## Summary

将画布升级为深度交互式工作台，用户通过选中卡片与系统对话，选择快速分析或深度探索模式，自定义卡片信息，并直接控制生成方向。

## Key Accomplishments

- **Card selection & basic interaction**: Double-click to select/deselect with persistent blue outline, hover-delete button with guards for source/child nodes, inline title editing for all node types.
- **Dialog refactor & generation control**: All chat context anchored to the selected card, direct generation of option nodes from dialog input with a "Generate" button, thinking/no-thinking mode toggle with API parameter passing and localStorage persistence.
- **Research mode & deep analysis**: "Research" hover-dropdown replacing the Analyze button with "Analyze" (no-thinking) and "Explore" (thinking) modes, explore mode gathers reference materials from the web, double-clicking option nodes with references opens a reference materials modal.
- **Image sharing**: Circular share button in the image viewer modal, standalone share-image page displaying the image, AI explanation, source summary, and generation direction, with one-click clipboard copy.
