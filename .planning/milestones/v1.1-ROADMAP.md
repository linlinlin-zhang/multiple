## Summary

将画布升级为智能创作工作台，支持多种输入格式、更高效的画布交互、完善的图片管理与个性化设置。

## Key Accomplishments

- **Rich multi-format input support**: Added DOCX/TXT/PDF/PPTX text extraction (zero-dependency heuristic parsing) and web link analysis with SSRF protection, all feeding into a unified AI analysis pipeline with source-type badges and full history browser integration.
- **Canvas interaction polish**: Denser grid, contained chat dialog styling, and refined navigation (plain scroll pans, Ctrl+scroll zooms) plus selective/full node collapse with multi-click controls and auto-collapse after generation.
- **Settings, dark mode, and i18n**: Database-backed per-role API configuration with runtime override without restart; system-wide dark mode with anti-FOUC scripts and cross-tab sync; lightweight bilingual i18n (Chinese/English) across both canvas app and React history browser with language-aware AI prompts.
- **Auto-arrange canvas layout**: One-click "Arrange" button that collapses un-generated option nodes and animates visible nodes into a clean layered tree layout using ease-out-cubic interpolation.
- **Image viewer modal with actions**: Clickable generated images open an accessible overlay modal showing enlarged image and AI explanation, with regenerate, custom-prompt modify, and direct blob download capabilities (including backend Content-Disposition support).
