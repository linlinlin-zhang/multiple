# ThoughtGrid / 织境

A multimodal AI canvas workbench with persistent sessions, material library, history browser, sharing, agents, research, and image/video generation.

## Project

- **Project docs**: `.planning/`
- **Roadmap**: 3 phases (see `.planning/ROADMAP.md`)
- **Current focus**: Phase 16 — Menu Cleanup & New Card

## Commands

```bash
npm run dev      # Start the server
```

## Tech Stack

- Node.js 18+ (ES modules, zero runtime deps for core)
- PostgreSQL + Prisma (persistence)
- React 19 + Vite 6 + Tailwind CSS (history browser UI in `app/`)
- Vanilla JS canvas app (`public/app.js`)
